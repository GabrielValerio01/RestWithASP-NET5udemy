HTTP Status Codes
1xx Informacionais
2xx Sucesso
3xx Redirecionamento
4xx erro de client
5xx erro de server

2XX SUCESSO
200 = ok
201 = created
202 = accepted
204= no content

4XX ERRO CLIENT
400 = bad request
401 = unauthorized
403 = forbidden
404 = not found

5XX ERRO DE SERVER
500 = internal server erro

 Criar um novo recurso usa o POST 
 Pra bucar ou ler os dado de um recurso usa o GET 
 Pra atalizar na maioria da vezes vai usar o PUT e em alguma vezes usa o PATCH
